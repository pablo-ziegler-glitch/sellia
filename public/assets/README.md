# Assets de Valkirja

Este directorio contiene imágenes placeholder en SVG (texto). Reemplazalas por tus fotos reales en
formato `.webp` o `.avif` para mantener buen performance.

## Archivos incluidos
- `placeholder.svg`: imagen placeholder para proceso/productos.
- `og-image.svg`: imagen de OpenGraph (recomendado 1200x630).
- `favicon.svg`: ícono básico.

> Tip: optimizá las imágenes con herramientas como Squoosh antes de subirlas.
